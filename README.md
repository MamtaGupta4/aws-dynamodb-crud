# 🚀 AWS DynamoDB CRUD Operations

![AWS](https://img.shields.io/badge/AWS-DynamoDB-orange?logo=amazonaws)
![Status](https://img.shields.io/badge/Project-Completed-brightgreen)

## 📌 Project Overview

This project demonstrates how to perform basic CRUD (Create, Read, Update, and Delete) operations in Amazon DynamoDB using the AWS Management Console.

---

## ☁️ AWS Service Used

- Amazon DynamoDB

---

## 🛠️ What I Did

- Created a DynamoDB table named **StudentTable**
- Configured **StudentID** as the partition key
- Inserted a new item into the table
- Viewed the stored item
- Updated the existing item
- Deleted the item from the table
